# EFORK-3 Published Reference Validation

This reference case reproduces the manufactured-solution terminal errors
reported for the three-stage EFORK method by Ghoreishi, Ghaffari, and Saad
(2023).  It is separate from any chaotic-system run so that solver validation
does not rely on attractor classification.

The archived source scripts in `03_integrators/sources/` were supplied by
Dr. Luis Gerardo de la Fraga (CINVESTAV Unidad Zacatenco) and were the basis
for adapting EFORK to the chaotic-system workflows.
