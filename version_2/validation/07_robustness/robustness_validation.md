# Validación de robustez actual

La robustez se evaluó para cada rama mediante continuaciones EFORK C
independientes. La rama de historial completo varía \(h\) y horizonte sin
truncar historia; la rama de ventana finita incluye además variación de
\(L_m\).

Esta etapa verificó persistencia geométrica bajo perturbaciones numéricas,
pero las referencias se descartaron después por retorno casi periódico. La
persistencia registrada no se interpreta como robustez de un atractor
caótico ni clasifica ocultedad.
